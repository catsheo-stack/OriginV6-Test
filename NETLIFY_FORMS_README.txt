Netlify Forms Ready Build

This package fixes Netlify Forms detection.

IMPORTANT:
This ZIP is ROOT READY. Upload/extract the contents of this ZIP directly to Netlify.
Do not put the files inside an extra OriginV6 parent folder during deploy.

What was added:
- __forms.html with a static hidden Netlify form
- property-pathway.html hidden form confirmed
- wizard.js submits to form-name: origin-property-pathway

After deploy:
1. Click Enable form detection in Netlify.
2. Redeploy this ZIP.
3. Open the live Netlify URL.
4. Complete and submit one property pathway test.
5. Go to Netlify > Forms.
6. The form should appear as: origin-property-pathway.

If it still does not appear, wait 1-2 minutes and refresh the Forms page.
