Netlify Forms Hard Fix

This ZIP contains:
- index.html at ZIP root
- property-pathway.html at ZIP root
- forms.html at ZIP root
- __forms.html at ZIP root
- A static Netlify form detector in index.html, property-pathway.html, forms.html, and __forms.html

Deploy instructions:
1. In Netlify Forms page, click Enable form detection.
2. Go to Deploys.
3. Drag THIS ZIP directly into Netlify deploy area.
4. Wait for deploy to finish.
5. Open the live site URL.
6. Visit /forms.html once.
7. Complete a pathway test submission.
8. Return to Netlify > Forms and refresh.

Expected form name:
origin-property-pathway

Important:
If Forms still does not appear, create a NEW Netlify site and deploy this ZIP there. Some existing projects may need a fresh deploy after form detection is enabled.
