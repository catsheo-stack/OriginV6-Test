# Origin Property Concierge - Multi-page Netlify Website

## Pages
- index.html
- buying.html
- selling.html
- finance.html
- conveyancing.html
- property-pathway.html
- thank-you.html

## Wizard
The full questionnaire is inside `property-pathway.html` and controlled by `assets/js/wizard.js`.

## Netlify Forms
The contact step posts to Netlify Forms using the hidden form named:
`origin-property-pathway`

After deploy, test the form and check Netlify > Forms.

## Deploy
Upload this whole folder or ZIP to Netlify manual deploy.
